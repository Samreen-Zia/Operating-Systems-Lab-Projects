# Operating Systems Lab Projects
This repository contains my academic lab work for the Operating Systems course at Fatima Jinnah Women University (FJWU).

---

## 📂 Lab 04: Shell Scripting
Focused on automation and file management using Bash scripts.
* **Tasks:** File deletion logic, Salary calculator, Fibonacci series, and Arithmetic operations.

## 📂 Lab 05: Process Management
Focused on system calls and process hierarchies using `fork()`.
* **Tasks:** Creating multiple child processes and multi-level process trees (P -> P1 -> P2).

## 📂 Lab 07: System Calls (I/O Operations)
Focused on low-level file handling and I/O using C system calls.

### **Part 1: read() & write()**
* **Task 1:** Program to read 15 characters from user input and display them.
* **Task 2:** Program to count and display the number of characters read by the system.

### **Part 2: open() & File Manipulation**
* **Task 1:** Appending data from one file (F1) to another (F2) using the `O_APPEND` flag.
* **Task 2:** Implementing a file copy utility to replicate contents from a source file to a destination file.
