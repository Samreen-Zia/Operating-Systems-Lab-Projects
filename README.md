# Operating Systems Lab Projects
This repository contains my academic lab work for the Operating Systems course at Fatima Jinnah Women University (FJWU).

---

## 📂 Lab 04: Shell Scripting
Focused on automation and file management using Bash scripts.

* **Task 1:** Script to test `rm` command and exit status.
* **Task 2:** Gross salary calculator (HRA calculation).
* **Task 3:** Addition of numbers using arguments.
* **Task 4:** Fibonacci series generator.

---

## 📂 Lab 05: Process Management (C Language)
Focused on system calls and process hierarchies using `fork()`.

* **Task 1:** Creating two child processes from a single parent (P1 & P2).
* **Task 2:** Implementing a multi-level process hierarchy (P -> P1 -> P2).
